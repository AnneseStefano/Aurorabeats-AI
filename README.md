## Overview

AuroraBeats AI is a school web project created as part of a web design course.
It is an AI-themed music blog built with WordPress, using the Astra theme and various plugins for layout, forms, and content management.

The site focuses on AI-generated and AI-themed music content, with a clean and modern design.

---

## Features

- AI-themed music blog layout
- Custom pages built with Gutenberg block editor
- Contact form integration (SureForms)
- Responsive and mobile-friendly design
- SEO-friendly structure
- Custom header and footer via Insert Headers and Footers plugin
- Analytics-ready setup

---

## Tech Stack

| Area | Technology |
|---|---|
| CMS | WordPress |
| Theme | Astra |

| Plugins | Ultimate Addons for Gutenberg, SureForms, Insert Headers and Footers, Akismet |
